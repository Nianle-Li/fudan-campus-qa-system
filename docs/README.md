- `关系模式初稿及说明.md`：关系模式、主键外键和设计依据。
- `规范性分析.md`：范式分析与冗余/异常说明。

- `阶段演示说明.md`：前端、后端、数据库联动演示步骤。
- `自我测试说明.md`：最终提交前的自动化测试、数据库验证和人工演示检查。
- `GitHub仓库整理说明.md`：分支策略、目录结构和交付物索引。
- `最终展示材料.md`：最后一次课演讲提纲、演示脚本和答辩要点。
- `最终提交说明.md`：最终可运行版本、功能清单和评审入口。
